
# Facebook Playground GraphAPI

## How to

1. Get **Access Token** at  https://developers.facebook.com/tools/explorer/
2. Save as FB_TOKEN env variable
3. Run the script


	export FB_TOKEN="..."
	python3 fb-read.py

## Features

- ```whoami()```
- ```comments()```
- ```toJSON()```
